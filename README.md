# Login Page
This project was bootstarpped with 
[Create React App](https://github.com/facebook/create-react-app) and
[Create React App Adding TypeScript](https://create-react-app.dev/docs/adding-typescript/).

## Teck Stack
- HTML5
- CSS3
- React JS
- TypeScript
- Material UI
- Formik
- Yup
- Axios
- React Testing Library
- Jest

## Before launch this application
Following requirements should be satisfied
1. Node.js is installed with v14.
2. Install all dependencies
```
npm install
cd server
npm install
```

## Run Application
```
npm start
```

## Run backend
```
npm start
```
